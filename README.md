# libre_musica

This project aims to help learning music theory in string instruments.

## Run it

> pnpm install

> pnpm dev

## Roadmap

- 404 page
- Select chords
- Note and chart
- Musical notation
- Play notes
- Record, save, load
- Representations
    - Circle of fifths
    - Circle of Fourths
    - Circle of camelot
    - Harmonic Table
    - Euler's Tonnetz
